# Get Number Units

**Syntax:**

`getunit(Number)`

Removes the units from a number, allowing you to work with the numbers alone.

### Examples

On this example, when you remove the units from `$20 per meal` you get `20`.

```deci live
stripunit($20 per meal)
==> 20
```
