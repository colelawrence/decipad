# Compound Rate

**Syntax**:

`compoundrate(Rate, Number of Periods)`

Calculates the compound rate over a specified number of periods based on a constant rate.

### Examples

```deci live
compoundrate(5%, 36 months)
==> 479.181613597186048%
```
