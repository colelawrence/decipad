# Get Year, Month, or Day from a Date

**Syntax**:

`pick(Date, [year/month/day])`

Extracts the year, month, or day from a given date.

### Examples

`pick(date(2050-Jan-01), year)`

```deci live
pick(date(2050-Jan-01), year)
==> 2050
```
