# Column Standard Deviation

**Syntax**:
`stddev(Table.Column)` <br />
`variance(Table.Column)`

Calculate the standard deviation of a column. You can use `variance()`as an alternative syntax:

### Examples

```deci live
stddev([1, 2, 3, 4])
==> 1.25
```
