# Blockquote

The blockquote component is a fundamental block in our editor. It serves the purpose of displaying text that needs the reader's attention and focus.
The component is a slate element component meaning that it requires some slate specific props to be passed to the component when it is being rendered.
This component will not be used outside of Slate.
